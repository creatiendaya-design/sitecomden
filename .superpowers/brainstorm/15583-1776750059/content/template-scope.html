¿Qué quieres poder customizar en los templates de producto?

El admin ya tiene el campo "Tipo de Página" (STANDARD / LANDING). Ahora hay que definir qué tan profundo va la customización.

A

Contenido por producto

Cada producto tiene sus propios bloques de contenido: imágenes del hero, secciones de beneficios, testimonios, colores de fondo. Configurable individualmente desde el admin del producto.

Ej: Producto A tiene hero rojo con foto X; Producto B tiene hero azul con foto Y

B

Configuración global del template

Los templates (STANDARD, LANDING) tienen ajustes globales: colores del tema, tipografía, layout. Todos los productos con LANDING usan el mismo diseño base.

Ej: En Configuración > Templates, defino que LANDING siempre usa color primario azul navy

C

Ambos — global + override por producto

Base global configurable, con opción de override por producto. Cada producto puede sobreescribir los defaults del template.

Ej: Default LANDING = azul navy. Producto A overridea a rojo. Producto B usa el default.
